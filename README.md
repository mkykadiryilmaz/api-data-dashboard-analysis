# API Veri Dashboard Analizi

## Proje Hakkında

Bu proje, API üzerinden alınan verilerin Google Sheets kullanılarak analiz edilmesi ve görselleştirilmesi amacıyla hazırlanmış beginner seviye bir veri analizi çalışmasıdır.

Proje kapsamında API'den çekilen veriler temizlenmiş, düzenlenmiş ve bir dashboard üzerinde görselleştirilmiştir.

Amaç, veri analizi sürecinde veri çekme, veri temizleme ve dashboard oluşturma adımlarını uçtan uca deneyimlemektir.

---

## Veri Kaynağı

Veriler public API üzerinden alınmaktadır ve güncellenebilir yapıdadır.

---

## Kullanılan Araçlar

* Google Sheets
* API Entegrasyonu
* Pivot Table
* Veri Temizleme
* Dashboard Tasarımı
